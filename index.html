IT ISSUE TRACKER APP

In this project I built a simple JavaScript application to record IT tickets and keep a track of IT tickets raised and the statuses of each ticket. The user is able to open a ticket, provide description of the issue, severity of the issue, provide name of who raised the ticket and assign the ticket to a user. Once the ticket is dealt with the user can change the status from open to closed. Data is saved on local storage. Bootstrap framework was used to provide the styling of the application.
